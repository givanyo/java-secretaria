## Como executar

- Clone o projeto;
- Execute o script localizado em `src/banco/script.sql` no MySQL;
- Use o XAMPP ou ferramenta semelhante para iniciar um servidor MySQL local na porta 3306.

Na IDE de sua preferência:
- adicione o driver JDBC localizado em `src/lib` ao Build Path do projeto;
- execute o arquivo `src/app/Main.java`.
